## 후기 & 새로 배운 것

- 도큐먼트 잘 읽으면 생소한 것도 할만 하구나
- 200만행 이상 데이터 처음 다뤄봤다.
  - 판다스 데이터프래임은 램이 꽉차면 일정 행 이상 읽지 못함
  - 미리 필터링해서 필요한 데이터만 읽어 램을 아낄 수 있을 듯
- 동적 변수선언 
  -  globals()['변수명_{}'.format(i)]]
